About Prompt Vault

Prompt Vault is a simple, clean repository for storing and organizing AI prompts. Built with simplicity in mind, it helps you quickly find and copy the prompts you need.

Features

  • Browse prompts by category (Writing, Coding, Marketing, Business)
  • Search through all prompts instantly
  • One-click copy to clipboard
  • Clean, distraction-free interface
  • Works on all devices

How to Use

1. Browse prompts by category or search for specific keywords
2. Click "Copy Prompt" to copy any prompt to your clipboard
3. Paste into your favorite AI tool (ChatGPT, Claude, etc.)
4. Replace the [PLACEHOLDER] text with your specific information

Adding Your Own Prompts

To add your own prompts, edit the JSON files in the prompts/ directory. Each category has its own file (writing.json, coding.json, etc.). Follow the existing format to add new prompts.

Open Source

This project is open source and available on GitHub. Feel free to fork, modify, and use it for your own prompt collection. No attribution required, but always appreciated!
